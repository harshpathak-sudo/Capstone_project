# Capstone Project
### Some guidelines to follow: 
- While initiating a new branch, please use the `feat/{task_name}` for the branch name. 
- When a branch task is complete, do merge it in `dev` after pulling from the dev first. 
- The branch architecure is `custom branch -> dev -> main`. 
- The commit message should begin using the objective keyword: `feat:`, `update:`, `fix:` , `merge:`, etc.
